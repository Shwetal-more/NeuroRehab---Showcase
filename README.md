🧠 NeuroRehab: AI-Driven Neuro-Rehabilitation Platform

NeuroRehab is a comprehensive digital rehabilitation ecosystem designed to transform neurological recovery through gamified therapy and AI-powered computer vision.

It bridges the gap between clinical supervision and home-based rehabilitation, enabling patients to recover independently while giving clinicians access to objective, data-driven insights.

🚀 Vision

Neurological rehabilitation often faces two major challenges:

Low patient engagement
Subjective progress tracking

NeuroRehab addresses this by:

🎮 Gamifying Therapy
Transforming repetitive exercises into engaging, interactive experiences
🤖 AI-Powered Validation
Using real-time computer vision to ensure exercises are performed correctly
📊 Data-Driven Decisions
Providing clinicians with measurable insights like range of motion, accuracy, and adherence

📈 Key Modules

🎮 1. Adaptive Therapeutic Game Suite

A collection of specialized rehabilitation games targeting different motor and cognitive functions.

Core Capabilities:

📷 Vision-Based Tracking
Detects:
Shoulder & neck movement
Hand and finger activity
Facial expressions and symmetry
🧠 Face Match Memory Game
AI-driven facial tracking to improve:
Muscle control
Reaction time
Memory
🔁 Real-Time Biofeedback Loop
Provides immediate visual feedback for:
Posture correction
Movement accuracy
Effort consistency



🩺 2. Physician Cloud (Doctor Dashboard)

A centralized dashboard for clinicians to monitor and manage patient recovery.

Features:

📊 Patient Overview
Track adherence and identify risk patterns
📉 Deep Analytics
Compare current performance with historical baselines
⚙️ Custom Therapy Plans
Assign personalized game levels and difficulty
👤 3. Recovery Portal (Patient Dashboard)

A patient-focused interface designed to encourage consistency and motivation.

Features:

📅 Progress Tracking
Daily logs, milestones, and recovery trends
🎓 Educational Hub
Tutorials and guided instructions for each exercise
🔐 Secure Authentication
Managed via Supabase with isolated patient profiles
